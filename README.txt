Easily integrate Respondly chat bot in the wordpress site.

== Description ==

This plugin will help you easily integrate Respondly chat bot in the wordpress site without any code knowledge. Respondly chat bot is for instant communication with on-site visitors and customers and enables prompt resolution to their questions and quotes.

Please register with [Respondly](https://www.respondly.com/registering/#register) first to get a key so that it will be used in the plugin to work properly. Once proper key is provided, Respondly chat bot scripts will be loaded on the wordpress site automatically so that no need to touch the code. For more information, please read https://www.respondly.com/privacy-policy/.


== Installation ==

Please follow the steps below to install the plugin. (Please register with Respondly first to get the key)
https://www.respondly.com/registering/#register

1. Upload `ai-chat-bot.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to "Settings" -> "General", put your Respondly key in the field - Respondly Key, which is at the bottom of the page
